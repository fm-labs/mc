# mc-app

## Tech Stack

- Tauri
- Typescript
- Vite
- Bun (javascript runtime/package manager/bundler)
- React
- React-router
- Tailwind CSS 
- Shadcn UI

## Quick Start

**Install dependencies**

```bash
bun install
```

**Run development server**
```bash
bun run dev
```

**Run development Tauri app**
```bash
bun run tauri dev
```


## Developer

### Add shadcn UI components

```bash
bunx --bun shadcn@latest add button
```
